# ai-chatbot
An interactive chatbot called 'Peter' that talks about AI with you

How to download:

Requirements: Pip3 and Python3
```
$ git clone https://github.com/HaiderZak/ai-chatbot.git

$ cd ai-chatbot

$ pip3 install -r requirements.txt

$ python3 peter.py

OR

$ py peter.py

```
